# FicWriter

## Error logs
If the app misbehaves, send the log file to the developer.

Log location on Windows:
- `C:\Users\<YourUser>\AppData\Local\com.admin.ficwriter\logs\FicWriter.log`

If the file is missing:
1. Launch the app.
2. Reproduce the problem.
3. Close the app and check the path above.

What to send:
- One file: `FicWriter.log`

What is inside:
- JavaScript errors, unhandled errors, and warnings.

# FicWriter

## Логи ошибок
Если приложение работает нестабильно, отправьте лог файл разработчику.
Где искать лог:
- Windows: `C:\Users\<YourUser>\AppData\Local\com.admin.ficwriter\logs\FicWriter.log`

Если файла нет:
1. Запустите приложение.
2. Повторите действие, при котором возникает ошибка.
3. Закройте приложение и проверьте путь выше.

Что отправлять:
- Один файл `FicWriter.log`.

Что в логе:
- Ошибки JavaScript (UI), необработанные ошибки и предупреждения.
